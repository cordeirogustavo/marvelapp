## Stack
**Front-end:** NextJs, Jest, Styled Components

## Run in Locally
Install *Marvel App* with yarn

```bash
  git clone https://github.com/cordeirogustavo/marvelapp
  cd marvelapp
  yarn && yarn dev
```
    
## Run Tests

To run the tests, run the following command

```bash
  yarn test
```
